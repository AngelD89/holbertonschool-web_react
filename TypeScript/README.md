# TypeScript Project

## Description

This project introduces **TypeScript fundamentals** and progressively covers more advanced concepts such as:

- Interfaces and type annotations
- Classes and inheritance
- Advanced types and type guards
- String literal types
- Ambient namespaces and declaration files
- Namespaces and declaration merging
- Nominal typing using branding

Each task is organized into its own directory and built using **Webpack** and the **TypeScript compiler**, following Holberton School requirements.

---

## Requirements

- Ubuntu 18.04 LTS
- Node.js
- npm
- TypeScript
- Webpack

All files:
- Use the `.ts` extension when applicable
- End with a new line
- Compile with **no TypeScript errors**
- Follow Holberton School style guidelines
